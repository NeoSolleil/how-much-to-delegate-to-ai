# AIエージェントは賢いより忠実──だから任せる範囲は人間が決める

AIコーディングエージェント（Claude Code）に「どこまで任せるか」を考えた記事と、その題材になったサンプルコード・図をまとめたリポジトリです。

Qiita 記事として公開したものの元データを置いています。

## この記事の要点

- AIエージェントは賢いというより**忠実**で、指示した範囲をそのままなぞる
- だから「どこまで任せ、どこは人間が握るか」の線引きが必要になる
- 線は「実害の大きさ × 戻しやすさ」で引く（任せる / 確認してから / 任せない）
- これは最小権限・確認ゲートといったセキュリティの発想と地続き

想定読者：AIエージェントをこれから使う / 使い始めた人。

## サンプルコードについて

`src/age.py` は、記事の題材にした「ありがちなバグ」です。
月は比較しているが**日を比較していない**ため、誕生日が「今月のこれから来る日」のときだけ満年齢が1歳ズレます。普段は正しく動くので気づきにくい、という例として使っています。

```bash
python src/age.py
```

実行すると、正しく動くケースと1歳ズレるケースが出力されます。
AIエージェントに「バグを直して」と投げる実験用の素材としてどうぞ。
